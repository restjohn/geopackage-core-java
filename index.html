The GeoPackage Core Java library was developed as part of the GeoPackage Libraries by the National Geospatial-Intelligence Agency (NGA) in collaboration with BIT Systems.

GeoPackage Core provides core functionality for GeoPackage Java and Android implementations of the Open Geospatial Consortium GeoPackage spec.

It is the core library of the GeoPackage Android SDK and GeoPackage Java library, both which can be found under OGC GeoPackage Implementations by the National Geospatial-Intelligence Agency.
